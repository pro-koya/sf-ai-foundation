お世話になっております。宮林です。

先日ご案内した SF-AI-Foundation について、5/19（火）に概要デモを実施します。

このツールは、Salesforce プロジェクトの以下の課題に対処することを目的に開発しました。

- 新規参画者が現状把握するために詳しい人の時間を取ってしまう
- ドキュメントの手動更新が追いつかず、実装と乖離する
- リリース前の手動作業の見落としリスク

解決策として、CLI（`sfai`）と Claude Code を組み合わせたツールを開発しました。知識グラフの自動構築・ドキュメント生成・差分分類・手動作業検出を自動化します。

5/19 のデモは 15 分程度で、画面共有しながら基本操作を見ていただく予定です。事前に概要資料（01-overview-for-members.md）とよくある質問（04-faq.md）をご確認いただけると当日スムーズです。

Windows 環境でのセットアップ手順（02-quickstart-windows.md）も添付しています。デモ後に試してみたい方は、事前にセットアップいただけますと幸いです。

ご都合や質問がございましたら、お知らせください。

宮林 幸也
